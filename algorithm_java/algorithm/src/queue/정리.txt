##### 큐(Queue)
1. 인터페이스
2. 한쪽 끝에서만 자료를 넣고 다른 한쪽 끝에서만 자료를 뺄 수 있는 구조
3. First In First Out(FIFO) 구조
4. 용어 
	- push : 큐에 자료를 넣는 연산
	- pop : 큐에서 자료를 빼는 연산
	- front : 큐의 가장 앞에 있는 자료를 보는 연산
	- back : 큐의 가장 뒤에 있는 자료를 본는 연산
	- empty : 큐가 비어있는지 아닌지를 알아보는 연산
	- size : 큐에 저장되어있는 자료의 개수를 알아보는 연산
5. 종류
	5-1) ArrayDeque
	5-2) LinkedList
	5-3) PriorityQueue
