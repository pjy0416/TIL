###### Set
1. 인터페이스
2. 중복 원소는 포함 X
3. 종류
	3-1) HashSet				<- 일반적으로 사용
	3-2) TreeSet				<- 순서가 중요한 경우 사용
	3-3) LinkedHashSet			<- 입력으로 넣은 순서가 중요한 경우 사용
	
##### HashSet
1. 해시 테이블을 이용해서 구현
2. 삽입/제거/삭제 연산의 시간 복잡도 = O(1)
3. 순서 보장 X

##### TreeSet 
1. 이진 검색 트리를 이용해서 구현
2. 삽입/삭제/제거 연산의 시간 복잡도 = O(log n)
3. 순서 보장 O 

##### LinkedHashSet
1. 해시테이블에 연결 리스트 이용해서 구현
2. 삽입/삭제/제거 연산의 시간 복잡도 = O(1)
3. 추가한(입력한) 순서 보장 O
